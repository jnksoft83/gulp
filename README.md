# Purpose of the Project

Developed for my wife, who is a web publisher.

# How to Use

```
Install Node.js and npm.
Install Gulp CLI globally: npm i -g gulp-cli
Navigate to the downloaded source folder.
Run the following commands:
> npm install
> gulp
```
